## Features

- **[Themes](https://andrewmcamp.github.io/ggaib/articles/themes.md)** —
  Publication, presentation, data-dense, and map variants
- **[Color
  Scales](https://andrewmcamp.github.io/ggaib/articles/color-scales.md)**
  — Discrete, continuous, and diverging palettes
- **[Labels](https://andrewmcamp.github.io/ggaib/articles/labels.md)** —
  Dollar, comma, and percent axis formatters
- **[Palettes](https://andrewmcamp.github.io/ggaib/articles/palettes.md)**
  — Direct access to brand colors
- **[Font
  System](https://andrewmcamp.github.io/ggaib/articles/fonts.md)** —
  Automatic font registration with bundled fallbacks

## Installation

``` r
# install.packages("devtools")
devtools::install_github("andrewmcamp/ggaib")
```
