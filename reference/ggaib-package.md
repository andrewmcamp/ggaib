# ggaib: ggplot2 Themes for the Annenberg Institute at Brown University

Provides ggplot2 themes and color scales following the visual identity
of the Annenberg Institute at Brown University. Includes publication,
presentation, map, and data-dense theme variants plus discrete,
continuous, and diverging color scales using the Institute's brand
palette.
